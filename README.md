DEMO - [https://kanban-frontend-plum.vercel.app/]

## Technologies/libraries that were used:

- Next.js;
- TypeScript;
- Redux + redux-saga;
- MUI;
- Axios;
- dnd-kit;
- date-fns;

## How to start a server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
